# Becoming - My Technical Journey

A beautiful static website showcasing my 2025 technical review and 2026 goals.

## Live Website

Visit the website at: [https://gausk.github.io/becoming](https://gausk.github.io/becoming)

## Contents

- **2025 Review** - Comprehensive review of courses, projects, and learnings from 2025
- **2026 Plan** - Goals, roadmap, and plans for the upcoming year

## Features

- Modern, responsive design
- Clean and intuitive navigation
- Mobile-friendly layout
- Beautiful gradient color scheme
- Interactive cards and animations

## Tech Stack

- Pure HTML5
- CSS3 with modern features (Grid, Flexbox, CSS Variables)
- No dependencies - just static files!

## Structure

```
becoming/
├── index.html      # 2025 Review page
├── plan.html       # 2026 Plan page
├── styles.css      # Shared stylesheet
└── README.md       # This file
```

## Customization

The website uses CSS variables for easy customization. Edit the `:root` section in `styles.css`:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* ... other variables */
}
```

## Responsive Design

The website is fully responsive and works great on:
- Desktop (1200px+)
- Tablet (768px - 1199px)
- Mobile (< 768px)

## Deployment

### GitHub Pages Setup

1. Push this repository to GitHub
2. Go to repository Settings → Pages
3. Source: Deploy from a branch
4. Branch: `main` (or `master`)
5. Folder: `/ (root)`
6. Save

Your site will be live at `https://gausk.github.io/becoming`
